# Vue 文本编辑器设计

## 目标

在空目录中创建一个基于 Vue 的单页文本编辑器，适配手机和 PC。编辑器支持文本与图片内容块，用户可以预览并生成 HTML，也可以转换为 Markdown。

## 范围

- 使用 Vue + Vite 实现纯前端应用。
- 支持标题、段落、引用、无序列表和图片块。
- 支持添加、选择、编辑、删除内容块。
- 支持本地图片上传并以浏览器 Data URL 预览。
- 支持 HTML 与 Markdown 两种实时输出，并提供复制按钮。
- PC 端使用三栏工作台：左侧块列表，中间编辑区，右侧预览/导出。
- 移动端使用单列布局，工具区、编辑区、预览区上下排列。

## 架构

应用由一个 Vue 单页组件承载交互状态，导出转换逻辑放在独立工具模块中，便于测试。内容块使用统一结构表示：`id`、`type`、`text`、`level`、`items`、`src`、`alt`。

## 数据流

用户操作更新 Vue 响应式块数组。当前选中的块驱动编辑表单，块数组通过计算属性转换为 HTML 和 Markdown。复制按钮读取当前输出并写入剪贴板，失败时显示明确状态。

## 错误与边界

- 图片未上传时显示占位状态，不生成无效图片标签。
- Markdown 与 HTML 转换会转义 HTML 特殊字符。
- 删除当前块后自动选择相邻块。
- 空列表项会被过滤，避免输出多余项目。

## 测试

为导出转换逻辑添加 Vitest 单元测试，覆盖文本、标题、引用、列表、图片、HTML 转义和空图片跳过。通过 `npm test`、`npm run build` 和本地预览验证。
